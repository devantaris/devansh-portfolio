# Directory Structure

- `/app`: Next.js App Router root. Contains globals.css, layout, page, and subroutes.
- `/components`: Root folder for React components.
  - `/components/sections`: Page sections (e.g. `AboutSection.tsx`).
  - `/components/ui`: Primitive and interactive UI parts (e.g. `contact-modal.tsx`).
- `/public`: Static assets (images, icons).
- `/hooks`: Custom React hooks.
- `/lib`: Utility functions (e.g. `utils.ts` for Tailwind merge).
- `/.gemini`: GSD & AI Agent configurations/workflows.
- `/.planning`: Codebase map and execution context (generated by GSD tools).

## Key Files
- `components/Hero.tsx`: Main animated hero component using GSAP/Framer Motion.
- `components/StarfieldBackground.tsx`: Heavy WebGL background optimized for scroll performance.
- `components/CustomCursor.tsx`: Canvas-based custom cursor logic.
